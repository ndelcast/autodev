2
Acquire semaphore + dispatch
Prend un slot du semaphore, marque le projet comme "running", lance une goroutine dédiée.
sem <- struct{}{} // acquire slot
running[project.ID] = true
go func() {
  defer release() // libère slot + marque projet idle
  executor.Process(ctx, gen)
  scheduler.processNext(project.ID) // lance le prochain ticket du même projet
}()

3
Marquer "running"
Met à jour la génération en base : status = running, started_at = now()
4
Construire le prompt
Assemble context.md + skills + ticket. Écrit prompt.md dans un dossier temporaire.
fichier contexts/{project}.md
fichier skills/{skill}.md

5
Créer le container Docker
Via Docker SDK : container éphémère avec l'image stack du projet. Monte workspace, prompt, output.
docker.ContainerCreate(ctx, &Config{
  Image: project.DockerImage,
  Cmd: ["autodev-exec", branch, ticket, prompt],
  Env: [ANTHROPIC_API_KEY, GITHUB_TOKEN, ...],
}, &HostConfig{
  Binds: [workspace, prompt.md:ro, output]
})
image: autodev-{stack}:latest
6
Démarrer et attendre
Démarre le container. La goroutine attend la fin (blocking). Le container exécute autodev-exec.sh.
docker.ContainerStart(ctx, containerID)
statusCh, errCh := docker.ContainerWait(ctx, containerID)

11
Release slot + processNext
Libère le slot du semaphore, marque le projet comme idle. Si d'autres tickets du même projet sont en queue, lance le suivant.

delete(running, project.ID)
scheduler.processNext(project.ID) // FIFO du même projet
